# Retine

# Description : 

Etude sur l'évolution de la couleur des yeux au cour du temps. Il a été étudié par exemple sur un échantillon homonègne de personnes au yeux bleu,marron et vert si au fur et à mesure des générations une couleur de yeux dominer.

# Tech : 

Le logiciel Netlogo et son langage.

# Compilation :

Ce projet utilise Netlogo un logiciel de simulation.
Pour tester la simulation il est necessaire d'installer Netlogo via le site officiel.
Ensuite il suffira de lancer le fichier chromosome.nlogo et d'executer depuis netlogo la simulation (flèche verte).
Aussi dans le dossier "rapport" vous pouvez trouver un pdf décrivant le résultat des recherches.
